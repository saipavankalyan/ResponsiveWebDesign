Market Lens

  • A Full-stack web application that enables users to analyse various insightful market trends through informative graphs.

Gator Taxi

  • An application for a ride sharing service to keep track of pending request rides.
  • Efficient implementation using Min heaps and Redblack trees implemented in Java.
  • Features include printing ride details, adding a ride request, processing a ride, updating the trip, cancelling the ride.

Bitcoin Miner

  • Actor model based simulation of Bitcoin mining in Erlang.
  • Ability to launch hundreds of thousands of processes to mine bitcoins with K leading zeroes in the computed SHA256 digest for a random message and can run on multiple working machines.
  • Able to mine 28 leading zeroes with a speedup of 4.5 on 8 logical core system and tested on 2 working machine.
